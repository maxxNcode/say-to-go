About SAY TO GO

SAY TO GO is a voice-powered teleportation application that allows you to explore locations around the world through 360째 street-level imagery.

How It Works

  1. Click the "SPEAK LOCATION" button
  2. Allow microphone access when prompted
  3. Say the name of any location (e.g., "Eiffel Tower, Paris")
  4. The app will find and display 360째 street-level imagery of that location

Features

  • Voice recognition for hands-free navigation
  • Global coverage with street-level imagery
  • Automatic fallback to nearby cities when direct imagery is unavailable
  • Beautiful, animated user interface
  • Works on desktop and mobile devices

Technologies Used

  • Web Speech API for voice recognition
  • OpenStreetMap Nominatim for geocoding
  • Mapillary API for 360째 street-level imagery
  • HTML5, CSS3, and JavaScript (ES6+)

About the Creator

Mark Froilan Lendio (maxxNcode) is a passionate developer who loves creating innovative web applications that blend technology with user-friendly experiences. With a keen interest in geospatial technologies and voice interfaces, Mark created SAY TO GO as an educational project to demonstrate the power of modern web APIs.

When not coding, Mark enjoys exploring new places, both virtually and in real life, which inspired the creation of this teleportation app.

Project Motivation

SAY TO GO was born out of a desire to make location exploration more accessible and fun. The project showcases how modern web technologies can be combined to create engaging experiences that were previously only possible with dedicated applications.
