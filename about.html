Project Overview

SAY TO GO is a voice-powered teleportation application that allows you to explore locations around the world through 360Â° street-level imagery. As a student project, it showcases how modern web technologies can be combined to create engaging experiences that were previously only possible with dedicated applications.

How It Works

  1. Click the "SPEAK LOCATION" button
  2. Allow microphone access when prompted
  3. Say the name of any location (e.g., "Eiffel Tower, Paris")
  4. The app will find and display 360Â° street-level imagery of that location

Features

  • Voice recognition for hands-free navigation
  • Global coverage with street-level imagery
  • Automatic fallback to nearby cities when direct imagery is unavailable
  • Beautiful, animated user interface
  • Works on desktop and mobile devices
  • Smart suggestions for better location searches
  • Automatic country detection without manual configuration
