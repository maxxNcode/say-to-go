Project Overview

SAY TO GO is a voice-powered teleportation application that allows you to explore locations around the world through 360° street-level imagery. It showcases how modern web technologies can create engaging experiences that were previously only possible with dedicated applications.

How It Works

  1. Click the "SPEAK LOCATION" button or type a location
  2. Allow microphone access when prompted (for voice)
  3. Say or type the name of any location (e.g., "Eiffel Tower, Paris")
  4. The app will find and display 360° street-level imagery of that location

Features

  • Voice Recognition — Speak any location name to teleport instantly
  • Text Input — Type a location as an alternative to voice
  • Global Coverage — Explore cities, landmarks, and locations worldwide
  • 360° Views — Immersive street-level imagery powered by Mapillary
  • Search History — Quickly revisit previously searched locations
  • Nearby Search — Automatically finds nearby locations with imagery
  • Responsive Design — Works on desktop and mobile devices

Technologies Used

  • Web Speech API — Browser-based voice recognition
  • OpenStreetMap Nominatim — Geocoding location names to coordinates
  • Mapillary API — 360° street-level imagery
  • HTML5, CSS3, JavaScript (ES6+) — Core frontend technologies
  • Mapillary JS SDK — Interactive 360° view rendering

Project Motivation

SAY TO GO was born out of a desire to make location exploration more accessible and fun. As a student project, it serves as a learning platform to explore modern web APIs and demonstrates the power of web development in creating innovative solutions.
